# StreamVault - Netflix Style Movie App

A recruiter-friendly JavaScript portfolio project inspired by modern streaming platforms.

## Features

- Movie cards rendered with JavaScript
- Search and genre filter
- Movie detail modal
- Watchlist saved with localStorage
- Dark/light mode toggle
- Responsive layout

## Technologies

- HTML5
- CSS3
- Vanilla JavaScript
- localStorage
- Responsive CSS Grid

## Portfolio Skills Shown

- DOM manipulation
- Event handling
- State management
- Reusable functions
- UI/UX design
- GitHub Pages deployment
